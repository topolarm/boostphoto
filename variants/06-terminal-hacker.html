BURST2GIF v2.0 — Lightroom Classic Plugin
Convert burst photo sequences into smooth MP4 videos
or animated GIFs — directly from Lightroom,
preserving every edit you've made.
USAGE:
1. Select photos in Lightroom
2. File > Plug-in Extras > Burst2GIF
3. Click Export

burst2gif --features
--format gif|mp4
MP4 & GIF Export
Export to either format. High-quality output up to
6K resolution. Optimized for web and client delivery.
--preview live
Live Preview
Real-time playback before export. Adjust timing,
crop, and see your Lightroom edits reflected instantly.
--fps auto|<number>
Auto-Detect FPS
Reads EXIF capture intervals. Calculates optimal
frame rate automatically. Manual override available.
--platform macos
Native macOS
Built for Apple Silicon & Intel. Fast, native
performance. Zero external dependencies.
--detect capture-time
Smart Capture Time
Intelligently groups burst sequences by timestamp.
Works with any camera brand. Select range or auto.
