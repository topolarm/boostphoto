$ --format gif|mp4

MP4 & GIF Export

Export to either format with a single flag. High-quality output up to 6K resolution, ready for web or client delivery.

$ --preview live

Live Preview

See your GIF play back in real-time before exporting. Adjust timing, crop, and edits on the fly.

$ --fps auto

Auto-Detect FPS

Reads EXIF capture intervals and calculates the optimal frame rate automatically. No manual guesswork.

$ --platform macos

Native macOS

Built for Apple Silicon and Intel. Fast, native performance with zero external dependencies.

$ --detect capture-time

Smart Capture Time

Intelligently groups burst sequences by capture timestamp. Select a range or let the plugin figure it out.
